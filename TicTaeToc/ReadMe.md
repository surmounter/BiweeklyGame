## TicTacToeGate 클래스
* 사용자가 이벤트를 발생시키면, 그에 대응되는 `TicTacToeController`에 있는 함수를 호출해줍니다.

## TicTacToeController 클래스
* MVC 패턴에서 Controller에 해당하는 부분이다.
* 들어오는 이벤트에 맞게 Model의 멤버변수를 업데이트한다.
* Model의 멤버변수를 조회해서 View에 전달하여 그래픽을 업데이트한다.
